Cat Crossover

This was my second project from my Character Design class in which we had to take three characters from different media and crossover them. I picked Mao Mao, Meowscarada from Pokemon, and Kit from Brawl Stars cause all three are cats.

Lambert Turnaround

For the first Character Design project we had to do a turnaround sheet for a character. This is Lambert, the main character from Cult of the Lamb

Throne Room of Bugs

This was the final project from character Design in which we had to create original characters for a scene. These are Princess Beatrice, Theodore, and a worker bee in their bug kingdom.

Ferb Fire Emblem

The third project from character design was to take characters from one media and change their genre, so I put Ferb in a fantasy series, specifically Fire Emblem.

Adventure Time Boat
